## 나의 풀이

```python
answer = list(range(1, 10001))

def bruteforce(n):
    r = n%10
    q = int(n/10)
    dn = n + r
    while q>0:
        r = q%10
        q = int(q / 10)
        dn+=r
    try:
        answer.remove(dn)
    except:
        pass

for i in range(10001):
    bruteforce(i)

for elem in answer:
    print(elem)
```

### 시간복잡도

O(N)

### 실제 수행시간

135 ~ 138 ms

## 좋다고 생각하는 풀이

```python
nonSelfNum = set()
SelfNum = set(range(1, 10001))

def d(n):
    n = n + sum(map(int, str(n)))
    return n

for i in range(10001):
    nonSelfNum.add(d(i))

SelfNum -= nonSelfNum

for elem in sorted(SelfNum):
    print(elem)
```

### 시간복잡도

O(N)

### 실제 수행시간

20.6 ~ 20.7 ms

======

자릿수 문자는 확실히 문자열로 취급해서 푸는게 훨씬 빠른 것 같다.
크게보면 시간복잡도는 같을지 몰라도 수행시간에서 차이가 많이 난다.

그리고 띄어쓰기가 없는 숫자로 된 문자열은 그냥 map 돌리면 자릿수별로 분리된다는거 잘 알아두면 좋을 것 같다.
